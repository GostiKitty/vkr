# Раздаточный материал ВКР — TherNest

Карта расчётных модулей + полный справочник формул цифрового двойника теплозащиты здания.

## Файлы

| Файл | Что это | Для чего |
|---|---|---|
| `TherNest-karta-modulej.svg` | Карта модулей и потока данных, **вектор** | Печать на ватман A1 (масштабируется без пикселей), правка в Inkscape/Illustrator |
| `TherNest-karta-modulej.png` | Та же карта, растр ~4200×2970 | Вставка в слайды PowerPoint/Google Slides |
| `TherNest-formuly.docx` | Все **194 формулы** как **редактируемые уравнения Word** (OMML), A4 | Открыть/править в Word, копировать формулы прямо в текст ВКР |
| `TherNest-formuly.html` | Те же 194 формулы с рендером KaTeX | Открыть в браузере → «Печать → Сохранить как PDF» = готовый справочник |
| `TherNest-formuly.md` | Те же формулы, исходный Markdown | Правка текста; повторная сборка .docx через `pandoc` |

> HTML тянет KaTeX с CDN — при первом открытии нужен интернет (дальше можно печатать в PDF офлайн).

## Структура карты (строгий ч/б, 5 тиров)

```
1 · МОДЕЛЬ И ПРЕПРОЦЕССИНГ:  BuildingModel → граф смежности · геометрия BIM · ограждение/материалы · контекст сетей
2 · СБОРКА ТЕПЛОВОЙ МОДЕЛИ:  зоны (C, G_инф/G_вент) + связи Вт/К + проводимости UA (×bridgeFactor)
3 · РАСЧЁТНЫЕ КОНТУРЫ:       А) RC-солвер: старт→итерация×12→отопление→баланс→поле/динамика
                            Б) СП 50: R_пр→k_ob→k_vent→β→q_быт,q_сол→q_от→Q_год/класс
4 · АНАЛИЗ:                  неопределённость (Монте-Карло, Morris) · калибровка (RMSE/CVRMSE → обратная связь)
5 · РЕЗУЛЬТАТЫ:              энергопаспорт, отчёт СП 50, графики, 3D-поле, thermal.worker
ЧТО ОТКУДА БЕРЁТСЯ:         геометрия / материалы / климат (СП 131) / мосты холода (СП 230) / СП 50 табл. Д.2
```

## Перегенерация (если правился реестр формул)

```bash
cd twin-desktop/apps/desktop
./node_modules/.bin/esbuild src/entities/formulas/registry.ts --format=esm \
  --outfile=../../../razdatka/.gen/registry.mjs
node ../../../razdatka/.gen/generate.mjs    # → TherNest-formuly.md + .html
node ../../../razdatka/.gen/build-svg.mjs    # → TherNest-karta-modulej.svg
```

Источник истины: `src/entities/formulas/registry.ts` (194 формулы), `core/thermal/*`, `norms/sp50_2024/*`.
